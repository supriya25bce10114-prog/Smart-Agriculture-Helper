# 1. Problem Statement

Modern agriculture faces several challenges, especially for small and medium-scale farmers who do not have access to continuous monitoring tools. Key environmental factors like soil moisture, temperature, and humidity greatly influence crop health and yield, yet these parameters are rarely tracked effectively.

The problem addressed in this project is the lack of an accessible, simple, and low-cost system that can:

Recommend appropriate crops based on environmental conditions 

Recommend to irrigate when the soil moisture is low

This project aims to serve as a beginner-friendly digital tool to assist farmers in making better decisions.


---

# 2. Scope of the Project

This project provides a basic Python-based Smart Agriculture Helper system that performs the following:

Recommends suitable crops based on environmental thresholds

Displays meaningful results to the farmer

Out of Scope / Future Scope:

Integration with real physical sensors

Machine learning-based disease detection

Cloud-based dashboards or mobile apps

Automated irrigation control hardware



---

# 3. Target Users

This project is designed for:

Students learning IoT, agriculture tech, or Python

Developers experimenting with smart farming concepts

Farmers seeking a simple digital assistant (after real sensor integration)

Researchers exploring rule-based agriculture systems
